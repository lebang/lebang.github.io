# 多余的话 (2020)

# 前言

 工作的原因，最近接触了不少的面试者。这其中有刚毕业工作2、3年的年轻人，也有已经工作6、7年，甚至更久的职场老人。很明显的一个事实是：当下的环境，确实有很多公司"关门"，或者是部门裁撤，导致相当一部分人处于失业状态。所以，假如你对现在的工作不是特别满意，在这个特殊时期，也不要贸然辞职，尽量"骑驴找马"。

# 心态

 聊到求职与面试，这其实是一个双向选择的过程。无论此时，你处于什么状态，都一定要保持谦逊、务实的心态来看待这件事情。失意不自卑，得意不自傲。或许会有挫折，但是不要气馁，大不了多面试几家，多总结失败经验，总会找到合适的工作。也或许你开局就很顺利，拿到了不错的offer，但是也尽量走出自己的舒适区，多去争取、了解，以便做出当下的最优解。

# 面试

 对于开发来说，面试主要在于技术，但是随着年限的增长，又不仅仅是技术。每个人在意的点都不尽相同，我们在打好基础的同时，一定要端正心态：那就是你不可能正确回答所有的问题。我曾经面试遇到一个哥们儿，做了3、4年，项目按他说流水上千万，但是他就知道拿开源库套套模板，基础知识"一问三不知"，项目照样上线了，没任何问题呀！总结你会的，突出表达出来，不会的是可以放心大胆的说不会，不丢人。

# 工作

 不要把工作当做所有，尤其是刚毕业的小伙儿，有些人很投入工作，心思也全在工作之上，并不是说这样不好，而是说在面对不如意的情况时，不要去强求非的怎样，自己该努力努力，保持心态平和。开开心心的不仅仅是自己，也会影响到周围人，反之，也一样。
